Core Functionality

Text Interaction

Advanced conversational capabilities with context awareness. Support for markdown formatting and detailed explanations via /think command.

Image Understanding

Process and analyze images for visual queries. Supports multiple image formats and maintains context in image-based conversations.

Document Analysis

Read and understand text documents, PDFs, and web pages. Convert web content to markdown for easy reading.

Audio Processing

Handle voice messages and audio files with metadata preservation. Support for various audio formats.

Instagram Media

Download photos, videos, and reels from Instagram. Options for compressed media or original files.

YouTube Features

Convert videos to MP3, extract transcripts in multiple languages, and maintain metadata with cover art.

Web Integration

Convert web pages to markdown, extract content, and process online resources seamlessly.

Context Management

Smart token management and history tracking for maintaining conversation context across sessions.

Advanced Features

Security & Access Control

Granular user access control with configurable permissions. Secure handling of sensitive data and API keys.

Smart History Management

Intelligent chat history tracking with automatic token management. Contextual understanding across message types.

Advanced File Processing

Support for multiple file formats including images, documents, audio, and video. Automatic format detection and handling.

Web-based Configuration

User-friendly configuration editor accessible via web interface. Real-time updates to bot settings and system prompts.

Error Recovery

Robust error handling with automatic retries for network issues and API rate limits. Graceful degradation of services.

Advanced Logging

Comprehensive user activity tracking including first/last seen dates, chat types, and interaction patterns.

Available Commands

/start

Initialize the bot with a welcome message and setup instructions

/help

Display comprehensive list of available commands and their usage

/clear

Reset chat history and context for the current user

/insta <url>

Download Instagram media as compressed format

/instaFile <url>

Download Instagram media as original uncompressed files

/ytb2mp3 <url>

Convert YouTube videos to MP3 with metadata and cover art

/web2md <url>

Convert webpage content to formatted markdown

/ytb2transcript

Extract YouTube video transcripts in multiple languages

/think <query>

Get detailed AI reasoning for specific queries

Installation Guide

1

Python Setup

apt update
apt install python3.11
2

Virtual Environment

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
3

Dependencies

Install ffmpeg for audio processing:

apt install ffmpeg
4

Environment Setup

Configure your environment variables:

export TELEGRAM_TOKEN_KEY='your-telegram-token'
export GEMINI_API_KEY='your-gemini-api-key'
5

First Run

./run.sh

Access the configuration editor at http://127.0.0.1:8080
